# Ponteiros e sizeof em C

O programa utiliza variáveis básicas e ponteiros para mostrar como funciona
o armazenamento de dados na memória, além de comparar o tamanho de tipos
primitivos e ponteiros usando o operador `sizeof`.

## Funcionalidades

- Exibição do tamanho em bytes de tipos básicos
- Exibição do tamanho em bytes de ponteiros
- Impressão de valores acessados via ponteiros

## Objetivo acadêmico

Projeto desenvolvido com fins educacionais para auxiliar no aprendizado
inicial sobre ponteiros e organização de memória na linguagem C.
